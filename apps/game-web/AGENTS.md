# game-web

## Tech Stack

- React 18 + Vite + TypeScript
- socket.io-client for WebSocket

## Project Modules

- React SPA with Vite default structure
- Socket service for real-time game communication

## Development Flow

1. `npm install` - Install dependencies
2. `npm run dev` - Start development server (port 5173)
